# Prestige Motors — Sitio Web

Sitio web premium para el taller automotriz **Prestige Motors**.

## Contenido

- `Prestige Motors.dc.html` — página web completa (Design Component)
- `support.js` — runtime necesario para renderizar el Design Component
- `uploads/prestige_motors_logo_opcion_3.png` — logotipo oficial

## Uso local

Abrir `Prestige Motors.dc.html` directamente en el navegador, o servir la carpeta con un servidor estático:

```bash
npx serve .
```

Luego abrir `http://localhost:3000/Prestige%20Motors.dc.html`.
